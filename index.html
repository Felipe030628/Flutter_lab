¿Qué es Flutter?

Flutter es un SDK de Google para crear aplicaciones nativas de alto rendimiento.

SDK

Kit de herramientas de desarrollo que permite crear aplicaciones de manera eficiente.

Comandos Eficientes

Herramientas de línea de comandos integradas que aceleran el flujo de trabajo y la compilación.

Desarrollo Mobile

Enfoque orientado a la creación de aplicaciones nativas para iOS y Android.

Dart

Lenguaje de programación optimizado para la UI, rápido y diseñado por Google para Flutter.

Widgets

La unidad fundamental de construcción de interfaces; todo en Flutter es un widget.

StatefulWidget

Widgets que mantienen un estado dinámico que puede cambiar durante la ejecución.

StatelessWidget

Widgets estáticos que no cambian su estado una vez construidos.
